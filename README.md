A modern, responsive image resizing web application built with **HTML**, **CSS**, and **JavaScript**. Resize images instantly using custom dimensions or predefined social media presets, preview changes in real time, and download high-quality resized images directly from your browser.

---

## ✨ Features

* Drag & Drop image upload
* Live image preview
* Custom width and height resizing
* Maintain aspect ratio
* Social media presets

  * Instagram
  * Facebook
  * YouTube
  * LinkedIn
  * Twitter / X
  * Website Banner
* Export images as PNG, JPG, or WEBP
* Adjustable output quality
* Original & resized image information
* One-click download
* Glassmorphism user interface
* Responsive on desktop, tablet, and mobile
* Client-side processing (No backend required)

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* HTML5 Canvas API
* FileReader API

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/KaustubhDeshmane/Image-Resizer-Pro.git
```

### Navigate to the Project

```bash
cd Image-Resizer-Pro
```

### Run

Simply open the **index.html** file in your preferred web browser.

No installation or additional dependencies are required.

---

## 📖 Usage

1. Upload an image.
2. Enter custom dimensions or choose a preset.
3. Select the output format.
4. Adjust image quality if required.
5. Click **Resize Image**.
6. Download the resized image.
